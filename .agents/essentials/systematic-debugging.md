# Depuração Sistemática
- Não tente adivinhar ou fazer alterações às cegas ao corrigir um bug.
- Primeiro, reproduza o problema e entenda a causa raiz.
- Formule uma hipótese e teste-a sistematicamente.
- Adicione logs temporários ou use ferramentas de depuração para coletar mais contexto, se necessário.
- Uma vez corrigido, remova qualquer código de depuração temporário e considere adicionar um teste para evitar regressões.
