# Kaizen (Melhoria Contínua)
- Sempre busque deixar a base de código mais limpa do que a encontrou.
- Identifique e sugira pequenas refatorações ou melhorias ao trabalhar em um arquivo.
- Procure por gargalos de desempenho, duplicação de código ou más práticas e corrija-os.
- Trate cada tarefa como uma oportunidade para melhorar a saúde geral do projeto.
