# Data Visualization

Esta regra dita a transformação de dados técnicos e métricas em apresentações visuais compreensíveis e impressionantes.

## Representação de Dados
- Utilize bibliotecas robustas (como D3.js ou ECharts no front-end, ou matplotlib/seaborn elegantemente configurados no Python) para apresentar dados analíticos.
- **Precisão:** Gráficos nunca devem distorcer escalas ou esconder dados essenciais. A clareza da informação é primordial.

## Sinergia e Conflitos
- **Diferenciação:** Apesar de visual, distingue-se de `algorithmic-art.md` e `canvas-design.md`. Visualização de Dados prioriza a **clareza da informação analítica** sobre a estética pura. Complementa perfeitamente as métricas extraídas via `python-observability.md`.
